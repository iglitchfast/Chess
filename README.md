# Chess Game in Python

This is a Python-based chess game built using **Pygame**. It allows you to play chess in two modes:  
- **Player vs Player** (local two-player mode)  
- **Player vs AI** (basic computer opponent using Minimax with heuristics)

The project focuses on clean GUI design, basic AI, and an interactive chess experience.

---

## 🛠 Features

- Graphical chess board using Pygame
- Drag-and-drop or click-to-move functionality
- Sound effects for move, capture, check, illegal moves, etc.
- Undo and reset buttons
- Move log panel
- Pawn promotion and castling support
- Check and checkmate detection
- Basic AI using minimax-based evaluation
